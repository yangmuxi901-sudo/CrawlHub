## 二、数据库表设计

### 2.1 扩展新闻表

```sql
-- 在现有 finance_news 表基础上扩展
ALTER TABLE finance_news ADD COLUMN tags TEXT;        -- 标签
ALTER TABLE finance_news ADD COLUMN sentiment TEXT;   -- 情绪（正/负/中性）
ALTER TABLE finance_news ADD COLUMN tickers TEXT;     -- 相关股票代码
```

### 2.2 新增公告表

```sql
-- 交易所公告表
CREATE TABLE IF NOT EXISTS exchange_announcements (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    exchange TEXT,           -- 交易所：SH/SZ/HK
    code TEXT,               -- 股票代码
    title TEXT,              -- 公告标题
    pub_date TEXT,           -- 发布时间
    category TEXT,           -- 公告类别
    link TEXT UNIQUE,        -- 链接
    pdf_path TEXT,           -- PDF 本地路径
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 研报表
CREATE TABLE IF NOT EXISTS research_reports (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    source TEXT,             -- 来源机构
    title TEXT,              -- 研报标题
    author TEXT,             -- 分析师
    pub_date TEXT,           -- 发布时间
    category TEXT,           -- 行业类别
    tickers TEXT,            -- 相关股票
    link TEXT UNIQUE,        -- 链接
    pdf_path TEXT,           -- PDF 本地路径
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

## 四、数据应用

### 4.1 舆情监控

```sql
-- 查询今日公告数量
SELECT exchange, COUNT(*) FROM exchange_announcements
WHERE DATE(pub_date) = DATE('now') GROUP BY exchange;

-- 查询持仓股票相关新闻
SELECT title, pub_date, source FROM finance_news
WHERE tickers LIKE '%600519%' OR tickers LIKE '%贵州茅台%';
```

### 4.2 关键词预警

```sql
-- 监控融资相关关键词
SELECT title, pub_date, source FROM finance_news
WHERE article LIKE '%融资%'
   OR article LIKE '%定增%'
   OR article LIKE '%可转债%'
   OR article LIKE '%配股%'
ORDER BY pub_date DESC LIMIT 100;
```

## 七、注意事项

1. **反爬策略**：交易所网站访问频率需控制
2. **PDF 存储**：公告 PDF 需要较大存储空间
3. **数据合规**：仅用于内部研究，勿商业用途
4. **监控告警**：爬虫失败时发送通知
