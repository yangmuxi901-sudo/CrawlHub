## 二、目标网站分析

### 2.1 隆众资讯 (oilchem.net)

**网站特点**：
- 行业专业数据平台
- 部分数据需要会员
- 页面多为动态渲染

**目标页面**：
1. 价格行情页：`https://www.oilchem.net/price/`
2. 开工率数据页：行业报告/周报中
3. 企业装置页：需要会员权限

**反爬策略**（预估）：
- 登录验证
- 频率限制
- 动态Token

### 2.2 备选数据源

| 网站 | 数据类型 | 难度 | 备注 |
|------|----------|------|------|
| 卓创资讯 (sci99.com) | 价格+开工率 | 高 | 付费为主 |
| 百川盈孚 (baiinfo.com) | 价格 | 中 | 部分公开 |
| Wind 终端 | 全量 | 低(付费) | 直接导出 |
| AkShare | 部分化工 | 低 | 开源库 |

## 三、技术要求

### 3.1 爬虫技术栈

**推荐方案**：Playwright（动态页面支持）

```
Python 3.10+
Playwright (浏览器自动化)
requests (简单页面)
pandas (数据处理)
```

### 3.2 代码结构

```
crawler-project/
├── crawlers/
│   ├── __init__.py
│   ├── base.py              # 基类
│   ├── oilchem_price.py     # 隆众价格爬虫
│   └── oilchem_utilization.py # 隆众开工率爬虫
├── parsers/
│   ├── __init__.py
│   └── oilchem_parser.py    # 页面解析器
├── storage/
│   ├── __init__.py
│   └── csv_exporter.py      # CSV导出
├── config/
│   ├── __init__.py
│   └── products.yaml        # 产品配置
├── output/                  # 输出目录
│   ├── chemical_prices.csv
│   └── chemical_utilization.csv
├── main.py
└── requirements.txt
```

### 3.3 爬取频率

| 数据类型 | 频率 | 说明 |
|----------|------|------|
| 产品价格 | 每日 | 交易日收盘后 |
| 开工率 | 每周 | 周报发布后 |

### 3.4 输出格式

**CSV 格式要求**：
- 编码：UTF-8
- 日期格式：YYYY-MM-DD
- 数值：保留2位小数
- 文件命名：`chemical_prices_YYYYMMDD.csv`

**示例输出** (chemical_prices.csv):
```csv
product_name,product_category,price,price_change,unit,region,trade_date,source
涤纶POY,化纤,8500.00,2.50,元/吨,华东,2026-02-26,隆众资讯
涤纶FDY,化纤,8700.00,1.80,元/吨,华东,2026-02-26,隆众资讯
PVC,塑料,6500.00,-0.50,元/吨,华东,2026-02-26,隆众资讯
```

## 四、接口约定

### 4.1 导入到 AI Hedge Fund

爬虫输出 CSV 后，使用项目的 ETL 流程导入：

```
爬虫项目输出 CSV
      ↓
data/chemical_prices.csv  (放到 data 目录)
      ↓
python scripts/import_chemical_data.py  (待开发)
      ↓
industry_metrics 表
```

### 4.2 数据库表结构

```sql
CREATE TABLE IF NOT EXISTS industry_metrics (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    ticker TEXT,
    trade_date TEXT,
    industry TEXT DEFAULT '化工',

    -- 化工专用
    product_spread REAL,        -- 产品价差
    spread_percentile REAL,     -- 价差分位
    capacity_utilization REAL,  -- 开工率
    inventory_days REAL,        -- 库存天数

    -- 原始价格数据
    product_price REAL,         -- 产品价格
    price_change REAL,          -- 涨跌幅

    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,

    UNIQUE(ticker, trade_date, industry)
);
```
