### 🗄️ 数据库表

```sql
-- 新闻表 (finance_news)
CREATE TABLE IF NOT EXISTS finance_news (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    title TEXT,
    source TEXT DEFAULT '财联社',
    pub_date TEXT,
    link TEXT,
    article TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    UNIQUE(link)
);

-- 索引
CREATE INDEX idx_news_pub_date ON finance_news(pub_date);
CREATE INDEX idx_news_source ON finance_news(source);
```

**查询示例**:
```bash
$ sqlite3 data/sync_record.db "SELECT source, COUNT(*), MAX(pub_date) FROM finance_news GROUP BY source"

财联社 | 50 | 2026-03-04 20:02:52
第一财经 | 66 | 2026-03-04 20:01:48
```

## 三、数据应用

### 1. 舆情监控

```bash
# 查询今日新闻
sqlite3 data/sync_record.db "SELECT source, COUNT(*) as count FROM finance_news WHERE DATE(pub_date) = DATE('now') GROUP BY source"
```

### 2. 关键词匹配

```bash
# 查找与持仓股票相关的新闻
sqlite3 data/sync_record.db "SELECT title, pub_date FROM finance_news WHERE article LIKE '%贵州茅台%' OR article LIKE '%600519%'"
```

### 3. 最新新闻

```bash
# 查看最新 10 条新闻
sqlite3 data/sync_record.db "SELECT source, title, pub_date FROM finance_news ORDER BY pub_date DESC LIMIT 10"
```

## 四、使用指南

### 运行单个爬虫

```bash
cd /Volumes/KESU-1/Data-Fin/股东报告
source venv/bin/activate

# 财联社电报
python crawlers/news_cls.py

# 第一财经
python crawlers/news_yicai.py
```

### 使用调度服务

```bash
# 立即执行新闻爬取
python scheduler_service.py --run-now news_cls
python scheduler_service.py --run-now news_yicai

# 启动调度服务（后台运行）
python scheduler_service.py
```

### 查看执行日志

```bash
tail -f data/scheduler.log
```

## 五、注意事项

1. **反爬策略**：部分网站需要添加 User-Agent 和请求间隔
2. **代理准备**：大规模爬取需要代理池
3. **数据合规**：仅用于个人研究，勿商业用途
4. **API 更新**：财联社 API 的 sign 参数可能变化，需定期检查
