## 三、推荐方案

### 方案 A：组合指标法（推荐，零成本）

用可观测指标构建**替代开工率指标体系**：

| 指标 | 来源 | 频率 | 与开工率关系 |
|------|------|------|-------------|
| 期货价格 | AkShare | 每日 | 供需反映 |
| 期货库存 | AkShare | 每日 | 反向关系 |
| 行业指数 | 同花顺 | 每日 | 景气度正相关 |
| 产品价差 | 计算 | 每日 | 利润驱动开工 |
| 原油价格 | AkShare | 每日 | 成本端影响 |

**优点**：
- 完全免费
- 可自动化
- 实时性强

**缺点**：
- 间接指标，非真实开工率

---

### 方案 B：研报 PDF 解析（半自动）

**流程**：
```
研报平台 → 下载 PDF → OCR/解析 → 提取表格 → 结构化数据
```

**技术栈**：
- `pdfplumber` / `PyMuPDF` - PDF 表格提取
- `pandas` - 数据清洗

**示例代码**：
```python
import pdfplumber

def extract_table_from_pdf(pdf_path):
    with pdfplumber.open(pdf_path) as pdf:
        tables = []
        for page in pdf.pages:
            table = page.extract_table()
            if table:
                tables.append(table)
    return tables
```

**优点**：
- 数据准确（来自期货公司研报）
- 覆盖主要产品

**缺点**：
- PDF 格式不统一
- 需要持续维护
- 版权风险

---

### 方案 C：行业协会 + 上市公司公告（手动校准）

**数据来源**：
1. 中国石油和化学工业联合会 - 月度报告
2. 各细分行业协会 - 开工率数据
3. 巨潮资讯网 - 上市公司公告中的装置信息

**优点**：
- 官方数据，权威
- 完全免费

**缺点**：
- 更新频率低（月度）
- 无法自动化

---

## 四、最终推荐

**日常监控**：方案 A（AkShare 期货价格 + 行业指数）

- 或开发 PDF 解析器半自动提取
